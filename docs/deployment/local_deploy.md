# Local Deployment

This project is deployable as static files.

## Local File Mode

Open:

```text
index.html
```

## Local Server Mode

```powershell
python -m http.server 4173 --bind 127.0.0.1
```

Then open:

```text
http://127.0.0.1:4173/
```

## External Deployment

No external deployment was performed in this step. If external hosting is requested later, revalidate public wording against the current package and keep the completed Codex Security report attached as release evidence.
