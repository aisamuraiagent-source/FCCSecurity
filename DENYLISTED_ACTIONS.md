# Denylisted Actions

- External scanning or probing.
- Exploitation, brute force, malware, persistence, evasion, or credential collection.
- Printing secrets, tokens, cookies, private keys, or private incident data.
- Network access without explicit approval.
- Package installation without explicit approval.
- GitHub writes, commits, pushes, PRs, issues, comments, or labels without explicit approval.
- CI/CD edits, hooks, background automation, deploys, or permission changes without explicit approval.
- Publishing raw local scan artifacts as public evidence without sanitization.
